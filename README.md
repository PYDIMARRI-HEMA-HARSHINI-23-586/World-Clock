# World Clock

A simple web application to display the current time in different cities.  
View it here: [Live Demo](https://pydimarri-hema-harshini-23-586.github.io/World-Clock/)

---

## How to Use

1. Clone the repository.  
2. Open `index.html` in your web browser.

---

## Features

- Displays the current time for **Los Angeles** and **Paris** by default.  
- Allows users to select a city from the dropdown to view its current time.  
- Option to display the time for the user's **current location**.  

---

## Technologies Used

- HTML  
- CSS  
- JavaScript  
- [Moment.js](https://momentjs.com/)  
- [moment-timezone](https://momentjs.com/timezone/)
